# Database Project

This repository contains the code, reports, diagrams, and documentation for the three-part database project.

## Project Description

The purpose of this project is to design and implement a relational database system based on the given case study. The project includes conceptual modeling, logical database design, normalization, SQL implementation, and embedded SQL using Python.

## Repository Structure

- `code/` contains the Python and SQL code used for the project.
- `reports/` contains the reports for Project Parts 1, 2, and 3.
- `diagrams/` contains ERD diagrams and other design visuals.
- `data/` contains database files or datasets used in the project.

## Project Parts

### Part 1
Includes the conceptual data model and ERD.

### Part 2
Includes the logical data model, relational schema, keys, constraints, and normalization.

### Part 3
Includes the SQL implementation, queries, views, and embedded SQL code.
